What is Next.js?

The folder-based system that turns your project structure into a working web app

The most magical thing about the web

Every time you type a URL and press Enter, something extraordinary happens in under a second. But have you ever wondered what's actually happening between that keystroke and the page appearing?

Four actors, one page load

When you visit a URL, four characters talk it out behind the scenes. Watch their conversation — it's the core of how every web app works.

U
User (You)

I want to see the dashboard — I typed the URL into my browser.

B
Browser

Got it — I'll ask the Next.js Server for the page at this URL.

N
Next.js Server

URL received. The folder structure says app/dashboard/page.tsx handles this route. Let me find that component.

D
Database

Here's the data for this user's dashboard — their name, recent activity, and settings.

N
Next.js Server

Data received. I'll combine it with the page component and send back the finished HTML.

B
Browser

HTML received! I'm rendering the page now so you can see it on screen.

?

The key insight: Next.js is the middleman that figures out what to show based on where your files are, not on complex configuration files.

The folder is the map

Here's the secret that makes Next.js special: you don't write complex routing logic. Instead, the App Router uses your folder names as the address map. Folder = URL. File = page.

app/ The main pages folder — your folder structure becomes the URL map
dashboard/ Becomes /dashboard in the URL
page.tsx The component that renders the /dashboard page
login/ Becomes /login in the URL
page.tsx The login page component
layout.tsx The shared shell that wraps every page (nav, footer)
page.tsx The homepage — renders at /
components/ Reusable UI building blocks shared across pages
lib/ Shared logic and utility functions
ℹ️
Why this matters for working with AI

When you tell an AI "add a Contact page," you need to know it goes in app/contact/page.tsx. Understanding the app/ folder structure is the first step to directing AI effectively.
